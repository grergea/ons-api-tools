# ONS API Tools

ONS CDN 플랫폼 API를 활용한 CLI 도구 모음입니다.

## Tools

| Tool | Description | Docs |
|------|-------------|------|
| SSL Certificate Workflow | 인증서 등록·갱신·도메인변경·검증·비교·목록조회 자동화 (권장 진입점) | [ssl/README.md](./ssl/README.md) |
| SSL API Manager | ONS CDN SSL API 저수준 CLI | [ssl/README.md](./ssl/README.md) |

## Requirements

- Python 3.10+
- requests

```bash
pip install requests
```
